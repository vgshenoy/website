# vigneshenoy.com

Source code for my personal website - [vigneshenoy.com](https://vigneshenoy.com).

## Stack

- [Nuxt 4](https://nuxt.com) - framework
- [Nuxt UI v4](https://ui.nuxt.com) - component library
- [Nuxt Content v3](https://content.nuxt.com) - content layer (Markdown)
- [Tailwind CSS v4](https://tailwindcss.com) - styling
- Statically generated and prerendered

## Setup

```bash
pnpm install
```

## Development

```bash
pnpm dev
```

## Build

```bash
pnpm build
pnpm preview
```
